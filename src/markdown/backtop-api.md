## BackTop API

|       参数       |          说明          |    类型     | 可选值 |      默认值       |
| :--------------: | :--------------------: | :---------: | :----: | :---------------: |
|      target      |   绑定滚动事件的对象   | ElementNode |   -    | document / window |
| visibilityHeight |      滚动高度阈值      |   number    |   -    |        300        |
|     duration     |      动画持续时间      |   number    |   -    |        250        |
|     onclick      | 点击后动画结束触发事件 |  Function   |   -    |         -         |
