## API

|   参数   |       说明       |   类型   |    可选值    |    默认值    |
| :------: | :--------------: | :------: | :----------: | :----------: |
| visiable |   是否显示弹框   | boolean  | true / false |    false     |
|  title   |     弹窗标题     |  string  |      -       |    "标题"    |
| message  |     弹窗内容     |  string  |      -       | "提示的内容" |
|   onok   | 确认按钮回调事件 | function |      -       |    ()=>{}    |
| oncancel | 确认按钮回调事件 | function |      -       |    ()=>{}    |
